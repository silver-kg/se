# 프로젝트 정의서

## 1. 프로젝트 명칭
**Preedom-가짜 일정으로 행동을 앞당기고, 자유를 만든다**

## 2. 개발 시스템 명칭 정의하기
가상 마감 일정 생성 기반 작업 수행 유도 및 행동 데이터 기록/분석 시스템

## 3. 개발 시스템
본 서비스는 사용자의 미루기 행동을 줄이기 위해 실제 마감 이전에 가상의 마감 기한을 생성하여 작업 수행을 유도하는 것을 목적으로 한다.
사용자가 작업을 등록하면 시스템이 자동으로 가짜 마감 일정을 생성하여 실제 마감 전에 작업을 수행하도록 유도한다.
또한 작업 수행 과정에서 발생하는 실제 행동 데이터를 기록하여, 사용자가 자신의 수행 패턴을 객관적으로 확인할 수 있도록 한다.
이를 통해 단순한 일정 관리가 아닌, 행동 유도와 자기 인식을 동시에 지원하는 시스템 구현을 목표로 하였다.

## 4. 개발 시스템에서 제공하는 핵심 기능

### 4.1 가짜 마감 일정 자동 생성 (핵심 기능)
사용자가 작업 일정을 등록하면 시스템이 실제 마감 이전의 적절한 시점에 가상의 마감 기한을 자동으로 생성한다. 이를 통해 사용자가 작업을 미루지 않고 사전에 수행할 수 있도록 한다.

### 4.2 작업 수행 기록
작업 수행 과정에서의 활동 시간, 시작 및 중단 시점 등을 기록하여 실제 수행 데이터를 저장한다.

### 4.3 집중 패턴 분석
사용자의 작업 수행 데이터를 기반으로 시간대별 집중도, 평균 수행 시간, 작업 완료까지 소요 시간 등의 패턴을 분석하여 제공한다.

### 4.4 계획 대비 실제 수행 비교
사용자가 계획한 수행 시간과 실제 수행 시간을 비교하여, 계획과 실행 간의 차이를 시각적으로 제공한다.

### 4.5 반복 행동 패턴 분석
사용자의 반복적인 작업 수행 데이터를 정량적으로 분석하여, 집중 지속 시간, 작업 중단 빈도 등 행동 패턴을 도출한다. 산출되는 지표는 다음과 같다.

- 평균 집중 지속 시간
- 작업 시작 후 초기 중단까지 걸리는 시간
- 작업별 평균 수행 시간
- 일정 시간 내 중단 발생 비율

이러한 분석 결과를 통해 사용자의 반복적인 행동 패턴을 도출하고 시각적으로 제공한다.

### 4.6 데이터 기반 일정 보조
과거 수행 데이터를 기반으로 작업별 예상 소요 시간을 제시하여, 사용자가 보다 현실적인 계획을 수립할 수 있도록 돕는다.
